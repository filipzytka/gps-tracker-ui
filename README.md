## GPS Tracker UI

Frontend for GPS Tracker App. It allows you to see GPS data on Leaflet map. Created using NextJS with TailwindCSS.

```bash
npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
